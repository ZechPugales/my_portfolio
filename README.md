## About

This portfolio website demonstrates my web development skills with modern HTML, CSS, and JavaScript. It includes animated sections, responsive design, dark/light theme toggle, and smooth navigation.

## Features

- Responsive layout for desktop and mobile devices
- Dark/light theme toggle with saved preference
- Animated project showcase
- Rotating client logos
- Contact form with basic validation
- Smooth scroll navigation

## Technologies

- HTML5
- CSS3 (Flexbox & Grid)
- JavaScript (ES6)
- Git & GitHub

## Installation

1. Clone the repository:

```bash
git clone https://github.com/ZechPugales/my_portfolio.git
```

2. Navigate to the project folder:
```bash
cd my_portfolio
```

3. Open index.html in your preferred browser.

## Usage

- Explore projects in the Work section.
- Toggle between dark and light theme using the switch in the header.
- View client logos and testimonials.
- Use the contact form to reach out.
